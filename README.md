CoreElec DTSs and DTBs
-----------------------
All CoreElec DTS files compiled to DTB files and published in **different formats** in the [release section](https://github.com/maaaaz/coreelec_dts_and_dtb/releases):
* **source** upstream DTS
* **full** DTS assembled by resolving dependencies
* **compiled** DTB
* **decompiled** DTS
